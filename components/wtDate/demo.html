默认模式

wtDate.init($(".time"));

小时模式

wtDate.init($(".time1"),{
                    todayBtn: false,
                    startView: 0,
                    minView: 0,
                    maxView:1,
                    format: 'hh:ii'
            });

年-月-日-小时模式

wtDate.init($(".time2"),{
                    todayBtn: false,
                    startView: 2,
                    minView: 0,
                    maxView:4,
                    format: 'yyyy-mm-dd hh:ii'
            });
